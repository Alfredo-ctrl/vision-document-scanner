# Vision Document Scanner

Herramienta de visión por computadora para el escaneo, recorte y procesamiento de documentos. Utiliza transformaciones de perspectiva y umbralización adaptativa para convertir fotografías de documentos en imágenes limpias, legibles y de alto contraste, simulando un escáner real.

## Tecnologías
* Python
* OpenCV
* Numpy

## Uso
Instalar las dependencias:
```bash
pip install -r requirements.txt
```

Ejecutar el script:
```bash
python scanner.py -i input_image.jpg -o output_image.png
```
